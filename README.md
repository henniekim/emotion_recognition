# Emotion Recognition

본 프로젝트는 FER2013 얼굴 데이터를 이용하여 얼굴 표정을 인식하는 것이 목표입니다.

코드 구현에 Stanford 대학교의 CS231n 강의와 tensorflow 예제를 참조하였습니다.
"AlexNet"의 네트워크의 사이즈를 줄여서 사용하였습니다.

추가적으로 "Hide-and-Seek"을 적용하여 정확도를 조금 향상시켰으며,
약 70%의 정확도를 보입니다.
